# Sign Language Recognition System

## 📌 Project Overview
This project is a real-time Sign Language Recognition System developed using Python, TensorFlow, OpenCV, and Deep Learning. The system recognizes hand gestures representing sign language alphabets through a webcam and predicts the corresponding letter.

## 🚀 Features
- Real-time hand gesture detection
- Sign language alphabet recognition
- Deep learning-based classification
- Webcam integration using OpenCV
- User-friendly interface
- High accuracy prediction

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn

## 📂 Dataset
The model was trained using an American Sign Language (ASL) dataset containing images of hand gestures representing alphabet letters.
